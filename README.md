# raitask-ui
采用Vue开发的任务调度系统（Raitask）前端部分

*********************************************************
### 安装依赖包
```
npm install
或
npm install --registry=https://registry.npm.taobao.org
```

### 运行项目
```
npm run serve
```

### 构建项目到dist
```
npm run build
```
